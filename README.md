# businesscard_recognition
시각장애인용 명함 인식 어플리케이션


## 🖥️ 프로젝트 소개
시각장애인들을 위한 명함 인식, 저장 어플리케이션입니다.
<br>

## 🕰️ 개발 기간
* 23.09.27일 - 23.12.20일

### 🧑‍🤝‍🧑 멤버구성
 - 팀장  : 이현준 - 파이프라인
 - 팀원1 : 오윤석 - OCR, 어플리케이션
 - 팀원2 : 조재호 - 명함 정보 Detection

## 📌 주요 기능
#### 사진 선택
- 사진 찍기
- 불러오기
#### 정보 인식
- 이름, 번호, 직책 Bounding box detection
- OCR Model Recognition
#### 정보 저장
- 핸드폰 전화번호부 연동
- 인식한 정보 저장
